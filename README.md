News App Description
A News Application that allows users to sign in using Firebase Authentication and browse the latest news through a live API.
The app updates news in real time and lets users add articles to their Favorites list.
Favorite news are saved permanently and remain available even after refreshing the app or reopening it.
Features:
User authentication with Firebase
Live news updates using an API
Add and remove news from Favorites
Favorites are saved and do not disappear
Simple and clean UI
